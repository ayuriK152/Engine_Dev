# Engine_Dev

## DirectX12를 사용한 엔진 개발

DirectX12를 사용한 자체 엔진. 2D, 3D 모두 지원하는 엔진을 완성하는 것이 궁극적인 목표.

자세한 개발 현황 등 실시간 개발 현황을 정리한 [노션 페이지](https://ayurik.notion.site/DirectX12-1c705915ec4b80a7bc4ec8dad84b8a1d?pvs=4)
